# BingBot

Just a project I created to play with using python to control a webbrowser. This earns all the daily points for Bing Rewards using user-agent controls in Firefox. Using a bot is against Microsoft's Terms of Service, so this is just a proof of concept. 

## Features

- Supports multiple email accounts
- Earns both desktop, Edge, and mobile points

## Firefox setup

- Need to check "Delete cookies and site data when Firefox is closed" option under the Privacy and Security settings

- Also need to OK closing multiple tabs

## Python setup

All the following steps are performed within terminal:
- To create a virtual environment:`python -m venv env/`
- To start virtual enviornment: `source env/bin/activate`
- To run script within virtual enviroment: `python BingBot.py`
Python will tell you what packages are missing and need to be installed.
